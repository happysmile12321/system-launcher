## 功能演示 GIF

### 工作流创建流程
```
1. 点击"新建工作流"
2. 拖拽组件到设计器
3. 配置组件参数
4. 保存并启用工作流
5. 查看执行结果
```

### 容器管理操作
```
1. 查看容器列表
2. 点击"启动"按钮
3. 实时查看容器状态
4. 查看容器日志
5. 执行容器操作
```
